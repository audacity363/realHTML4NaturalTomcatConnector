How to use
Use the Protocol Extension tab only if you develop a macro control and do not provide a tag handler class. In this case, you can use this tab to append the referred properties and methods to the generation protocol. If you provide a tag handler class, you implement the addition of properties and methods to the generation protocol in the tag handler class instead.

Properties

  • Name = The name of the adapter property that is used within your control (mandatory).
  • Data Type = The data type of the property (optional). This information is used in the Code Assistant of the Layout Painter to generate appropriate property coding. (This has no meaning for custom controls in NATPAGE layouts.)
  • Preset Value = The value that is set in your adapter when the page is loading (optional).
  • Show in Code Assistant = When selected (default), this property appears in the Code Assistant.

Methods

  • Name = The name of the adapter method (mandatory).
  • Show in Code Assistant = When selected (default), this method appears in the Code Assistant. (This has no meaning for custom controls in NATPAGE layouts.)

JavaScript Libraries

  • Source = The URL that points to your JavaScript library (mandatory).
