How to use

Embedding Containers
In order to integrate your control into the control hierachy within the Layout Painter, you enter the list of container controls into which your control can be inserted as a subnode.

Example: You want to be able to insert your control into the row containers ITR and TR. To do so, you create the first line using the Add () button and specify "itr". Then you create the second line and specify "tr". If you want to delete lines from the list, you use the Delete () button.

  • Control Name = The name of the container control (mandatory). If specify your own controls, do not forget to use the library prefix.

Sub Controls
Your control can be a container itself. In this case, you enter the list of the subcontrols.

Example: You want to be able to insert the standard ICON control into your container. To do so, you create a line using the Add () button and specify "icon". If you want to delete lines from the list, you use the Delete () button.

  • Control Name = The name of the control that can be inserted into your container control (mandatory). If you specify your own controls, do not forget to use the library prefix.
