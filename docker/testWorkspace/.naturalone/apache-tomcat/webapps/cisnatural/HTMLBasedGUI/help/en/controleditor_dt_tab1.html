How to use
You use the Add () and Delete () buttons to specify the list of valid values. The column Id is mandatory. The column Description is optional.
Some Predefined Data Types ...
  • align (left|center|right)
  • boolean (true|false)
  • flush (screen|server)
  • height (some integer and percentage values)
  • scroll (auto|scroll|hidden)
  • textsize (1|2|3|4|5|6)
  • valign (top|middle|bottom)
  • width (some integer and percentage values)
  • wrap (soft|hard|off)
