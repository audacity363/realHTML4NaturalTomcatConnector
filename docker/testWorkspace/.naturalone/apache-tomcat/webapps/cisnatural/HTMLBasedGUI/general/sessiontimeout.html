Session Time Out

The session you worked in is not available anymore. Please relogon.

Possible reasons are:

  • You did not interact with the system for a certain period of time.   

    Dependent on your server setup sessions with no user interaction are automatically removed after a certain period of time.

  • You pressed the back button in the browser.

  • Your session was removed by a system adminstrator.

  • In development mode this message might also occur if working on several layouts simultaneously and switching between them.

 

 

 

 

Please contact your system administrator for more details.
